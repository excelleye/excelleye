# **HEAVEN'S GATE**

昔々あるところに  
何も分かたれず、色のない情景だけが広がる  
冷たい世界がありました。  

しかし、あるとき――  
空に大きな穴が現れました。  

穴の中からは、１匹の竜がやってきます。  

それは、枝分かれした大きな翼と、捻れた角を持つ、白い肌の竜でした。  

竜は、自分に流れる赤い血を使って、  
世界を真っ赤に染め上げます。  

真っ赤に染まった世界からは、  
幾匹もの命が誕生しました。  

血から生まれた、小さな命は  
自分たちを生んだ竜に、こうお願いします。  
「我らの王になってはくれないか？」  

ところが、竜はこう答えました。  
「残念だが、私は王になれない。その代わり、私の知る最も屈強な王を見繕ってやろう」  

“彼女”は自分に流れる血の中から鉄の鎧を取り出すと、それに魂を与えました。  

「我らの半身たるニムロッド王よ、臣下と共に手を尽くし、私を封印するといい」  

彼女はこう続けます。  

「そして、臣下たちに告げておく。封印は絶対に解かないことだ。もし封印が解かれたならば、私は全てを喰らうだろう」  

鉄鎧のニムロッド王は、自身の魂を星屑に変えると、天の大穴を星空の向こう側へと封じました。  
王の臣下たちは、ヘヴンズゲート（天国の門）を作り、地下深くの霊廟に竜を封じました。  

こうして、王が治める時代が始まったのです。  
果たして彼らが導く運命に、何が待っているのでしょうか？

# **Osborne - The Gloom Ones**

暗澹とした、深い霧の夜。  
誰も知らぬ、存在しない廃村で、  
ある財団が、ある教父の墓を暴いた。  
そして、無数の写本を発見する。  
人ならぬ智慧、神祖伝来の禁字教典。

>*The offering is taken to the altar.*  
（捧げ物は祭壇へ）  
*Leaf litter, herring guts, goat's eyeballs, and whale placenta.*  
（腐葉土、ニシンのはらわた、山羊の眼球、クジラの胎盤）  
*And listen to the voice at the bottom of the water.*  
（そして、水底の声を聴け）  
*Eventually, the entrails rot away and we finally encounter each other.*  
（やがて心臓は腐り落ち、我らは見える）  
*Rotting and stagnant, we finally encounter each other.*  
（腐り、澱み、遂に見える）  
*Oh, the Great gloom ones!*  
（ああ、暗澹なるものたちよ！）  

そこに記された羅列は、  
一体、何を意味するのか。  

財団は、  
知るべきでないことを調べ始めた。

# **夢見るステートマシン - The Dreaming State Machine**

ここではない何処か、今ではない何時か。  
誰かが望み、そして創った。  
幾多の断片が混ざり合う、奇妙な世界。  
雲上の大都市、トラキアへようこそ。  

貴方が求めるものは、平和で温かい日常？  
それとも、刺激的で冷たい非日常？
