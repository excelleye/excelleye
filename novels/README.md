# **HEAVEN'S GATE**



**戦王ニムロッド**の戦没が知れ渡った夜  
その遺骸を巡る争いと共に  
**古都エンデランド**は死地と化した。  
ニムロッドに連なる、**魂を持たぬ者**の時代は終わりを告げ  
彼らの伝説、その**封印**が崩壊を始めたのだ。  

その光景は凄惨たるものであったという。  
天蓋は形を失い、**星屑**となって地上に降り注ぎ――  
天に生じた大穴からは、**赤い血**の奔流が押し寄せた。

**星屑**は**魂**を、**赤い血**は**渇きの呪い**をもたらし  
やがて、**魂を持つ者**の時代が始まった。

# **Osborne - The Gloom Ones**

暗澹とした、深い霧の夜。  
誰も知らぬ、存在しない廃村で、  
ある財団が、ある教父の墓を暴いた。  
そして、無数の写本を発見する。  
人ならぬ智慧、神祖伝来の禁字教典。

>*The offering is taken to the altar.*  
（捧げ物は祭壇へ）  
*Leaf litter, herring guts, goat's eyeballs, and whale placenta.*  
（腐葉土、ニシンのはらわた、山羊の眼球、クジラの胎盤）  
*And listen to the voice at the bottom of the water.*  
（そして、水底の声を聴け）  
*Eventually, the entrails rot away and we finally encounter each other.*  
（やがて心臓は腐り落ち、我らは見える）  
*Rotting and stagnant, we finally encounter each other.*  
（腐り、澱み、遂に見える）  
*Oh, the Great gloom ones!*  
（ああ、暗澹なるものたちよ！）  

そこに記された羅列は、  
一体、何を意味するのか。  

財団は、  
知るべきでないことを調べ始めた。

# **夢見るステートマシン - The Dreaming State Machine**

ここではない何処か、今ではない何時か。  
誰かが望み、そして創った。  
幾多の断片が混ざり合う、奇妙な世界。  
雲上の大都市、トラキアへようこそ。  

貴方が求めるものは、平和で温かい日常？  
それとも、刺激的で冷たい非日常？
